SwapMonitor is a lightweight macOS menu bar utility for developers.

Frequently Asked Questions

Q: Does SwapMonitor collect any data?
A: No. SwapMonitor runs entirely offline and does not collect any personal information or usage data.

Q: How does SwapMonitor detect orphan processes?
A: It identifies processes with PPID = 1 that match known dev tool patterns (node, npm, vite, webpack, etc.).

Q: Is SwapMonitor safe to use?
A: Yes. It only reads system information and process data using native macOS APIs. It never sends data anywhere.

Q: How often does SwapMonitor update?
A: By default, it refreshes every 5 seconds. You can adjust this in Settings.
